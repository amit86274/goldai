# Gold-AI v1.0

AI-assisted trading framework for XAUUSD.

## Structure
- app/
- mt5/
- features/
- news/
- models/
- strategy/
- backtesting/
- dashboard/
- alerts/
- api/
- database/
- utils/

## Run

```bash
pip install -r requirements.txt
python app/main.py
```

## Docker

```bash
docker compose up --build
```
